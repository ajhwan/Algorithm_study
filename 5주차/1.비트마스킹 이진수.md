# 이진수
- 비트마스킹을 이해하기 위해서는 이진수를 이해해야 함
- 0 과 1, 두개의 숫자로 표현하는 이진법으로 표현하는 수가 이진수
- 일반적으로 이진법의 수를 십진법의 수와 구별하기 위해 다음과 같은 방법을 사용
  - 100101b (b를 덧붙임(b는 binary의 약자))(binary = 이진)
  - 100101(2) ((2)를 덧붙임, 주로 수학에서 쓰임)
  - 0b100101 (앞에 0b를 덧붙임)
