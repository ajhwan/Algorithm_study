# 너비우선탐색(BFS, Breadth First Search)
- BFS는 그래프를 탐색하는 알고리즘이며 어떤 정점에서 시작해 다음 깊이의 정점으로 이동하기전 현재 깊이의 모든 정점을 탐색하며 방문한 정점은 다시 방문하지 않는 알고리즘
- 같은 가중치를 가진 그래프에서 최단거리알고리즘으로 쓰임
- BFS로 탐색을 한다는 것은 이런식으로 레이어별, 레벨별로 탐색한다는 뜻
